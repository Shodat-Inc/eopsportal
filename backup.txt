## Prerequisites

npm install npm@latest -g

## Installation

1. Clone the repo  
   git clone https://github.com/Shodat-Inc/eopsportal.git.

2. Install NPM packages
   npm install

3. In your local env file,change the content from .env.Example for database connection with your local.

## Getting Started

3. Run the command accordingly

```bash
(if node)
npm run dev

(if yarn)
yarn dev

pnpm dev
```

## next.config

After setting up project,

1. First do isDBSync:true for the first time after setting up the project.
2. Then hit the API of "Create User" as according to the flow, first task is to create user.
3. When the response is saved and operation is done , do isDBSync:false.
